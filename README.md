## Стек технологий

Python • Aiogram • PostgreSQL • SQLAlchemy • Redis • NumPy • Pandas • Matplotlib

## Сборка

1. Создайте файл окружения `.env` на основе `.env.example`
2. Соберите и запустите контейнеры Docker: `make up`
